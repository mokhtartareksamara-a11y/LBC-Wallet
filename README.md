# LBC Wallet — Solana-Native Digital Wallet

> Your gateway to the LBC Digital City economy.

LBC Wallet is the official wallet for the LBC ecosystem — built on Solana for fast, low-cost transactions with native $LBC token support.

- Solana-native wallet
- $LBC token support
- USDC payments
- LBC ecosystem integration (Hub, Auto, Live, Protocol)
- Mobile-first design
